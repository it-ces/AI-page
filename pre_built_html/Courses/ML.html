Linear regression

The main is find the better line that adjust to the data, for instance for the following pictures constructed using the world bank data.

the problem in each datasets consist of a linear approximation \(\hat{y}_{i}\) over the real value \(y_{i}\) in which the error is measured by \(u_{i}^{2} = (y_{i} - \hat{y}_{i})^{2}\)

\begin{equation} \min_{\beta_{0}, \beta_{1}} \sum u_{i}^{2} \end{equation}

Here we go see that the values \(\beta_{0}\) and \(\beta_{1}\) that solve the problem could be gotted by two methods Maximun likelihood estimation and Least Square Method both methods offer similar results.
